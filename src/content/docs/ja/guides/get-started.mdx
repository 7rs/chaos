## ダウンロード  

  <Tabs>
    <TabItem label="HTTPS">
      ```sh
      git clone https://github.com/7rs/chaos.git --depth 1
      ```
    </TabItem>
    <TabItem label="SSH">
      ```sh
      git clone git@github.com:7rs/chaos.git --depth 1
      ```
    </TabItem>
  </Tabs>

  HTTPSかSSHでChaosのリポジトリをクローンします。  
  オプションは無くても問題無いですが、転送量が減って速く終わるので、使うべきです。  

  <LinkCard
    title="Git - git-clone Documentation"

## ビルド  

  ```sh
  packwiz modrinth export
  ```  

  `cd chaos`などをしてChaosのルートディレクトリに移動し、exportコマンドを実行します。  

  ```sh
  packwiz modrinth export --output dist/chaos.mrpack
  ```  

  出力先やファイル名を変えることもできます。  

  <LinkCard
    title="packwiz modrinth export"
    href="https://packwiz.infra.link/reference/commands/packwiz/modrinth/export"

## インストール  

  インストールは簡単です。
  クライアント側なら、[Prism Launcher][prism-launcher]や[Modrinth App][modrinth-app]などを使います。  
  サーバ側なら、[packwiz-installer][packwiz-installer]や[mrpack-install][mrpack-install]などを使います。
