## Download  

  <Tabs>
    <TabItem label="HTTPS">
      ```sh
      git clone https://github.com/7rs/chaos.git --depth 1
      ```
    </TabItem>
    <TabItem label="SSH">
      ```sh
      git clone git@github.com:7rs/chaos.git --depth 1
      ```
    </TabItem>
  </Tabs>

  Clone repository of Chaos with HTTPS or SSH.  

## Build  

  <Tabs>
    <TabItem label="Default">
      ```sh
      packwiz modrinth export
      ```  

      Move to the root directory of Chaos with `cd chaos` and Run the export command in there.
    </TabItem>
    <TabItem label="Specify output">
      ```sh
      packwiz modrinth export --output dist/chaos.mrpack
      ```  

      You can also change the output path and the file name.  

      <LinkCard
        title="packwiz modrinth export"
        href="https://packwiz.infra.link/reference/commands/packwiz/modrinth/export"
      />
    </TabItem>
  </Tabs>

## Installation  

  Installation is easy.  
  In client-side, use [Prism Launcher][prism-launcher], [Modrinth App][modrinth-app] or etc.  
  In server-side, use [packwiz-installer][packwiz-installer], [mrpack-install][mrpack-install] or etc.
